# 大學專題#2
此為菜鳥時的大學作品
## 利用遊戲的方式，學習單字



**創意想法:**  

透過新世代的科技和網際網路的發達，學習領域中，開創出各式各樣的行動英文學習APP，

行動英文學習APP不但讓學生學習不在侷限於教室裡，且可以利用零碎的時間進行學習，

進而幫助學生學習英文字彙及提升學習成效。


**後臺管理:**  
使用 SQL 錯誤單字，為後臺管理。
